# Ruang Teman - Unsoed Student Support

Ruang Teman adalah platform *web-based* yang dirancang khusus untuk mahasiswa Universitas Jenderal Soedirman (Unsoed). Platform ini bertujuan menjadi ruang aman dan suportif bagi mahasiswa untuk berdiskusi, bertukar pengalaman, dan mendapatkan panduan terkait kehidupan akademik, organisasi, hingga persiapan karier.

## Fitur Utama
* **Tanya Senior:** Forum tanya-jawab seputar akademik, magang, organisasi, dan karier dari pengalaman senior.
* **Ruang Cerita:** Wadah untuk berbagi keluh kesah dan cerita perkuliahan secara anonim atau publik.
* **Self Assessment:** Fitur evaluasi diri untuk mengetahui minat dan potensi karier mahasiswa.
* **Roadmap:** Panduan langkah demi langkah untuk mencapai target akademik dan karier.
* **Artikel & Bantuan:** Kumpulan informasi, tips, dan pusat bantuan terpadu.
* **Dashboard Mahasiswa:** Halaman personalisasi untuk memantau aktivitas dan interaksi.

---

## Teknologi yang Digunakan
Proyek Frontend ini dibangun menggunakan teknologi modern:
* **HTML5:** Struktur semantik website.
* **Tailwind CSS:** Framework CSS untuk *styling* yang cepat, responsif, dan mendukung efek modern (seperti *Glassmorphism*).
* **JavaScript (Vanilla):** Interaktivitas halaman, simulasi *database* (Local Storage), dan manajemen DOM.
* **Netlify:** Layanan *cloud hosting* untuk *deployment* berkelanjutan (CI/CD).

## Cara Menjalankan Proyek di Komputer Lokal
Jika Anda ingin menjalankan atau mengembangkan proyek ini di komputer Anda:

1. Lakukan *Clone repository* ini:
   ```bash
   git clone [https://github.com/hanifahsilmifajri-cloud/ruangteman-unsoed.git](https://github.com/hanifahsilmifajri-cloud/ruangteman-unsoed.git)
2. Buka folder proyek di *code editor* (direkomendasikan Visual Studio Code).
3. Pastikan ekstensi **Live Server** sudah terpasang di VS Code.
4. Buka file `index.html` (yang berada di luar folder utama), klik kanan, dan pilih **Open with Live Server**.
5. Website akan otomatis terbuka di browser Anda (biasanya di `http://127.0.0.1:5500`).
```
